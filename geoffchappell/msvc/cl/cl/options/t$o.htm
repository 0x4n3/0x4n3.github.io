CL /To

Syntax

/Tofilename

The required argument consists of whatever follows immediately within the same command-line token, else is the whole of the next command-line token.

Failure to provide an argument is an error (D2004). There is no formal constraint on the length of the argument.

Relations

This option has no syntactic relations with other options: it does not override any; it is not incompatible with any; it has no prerequisites.

Behaviour

The sole action of this option is to add filename to the list of input files, specifically as an object file.

Documentation Status

This option does not appear among the Compiler Options Listed Alphabetically in the product documentation.
