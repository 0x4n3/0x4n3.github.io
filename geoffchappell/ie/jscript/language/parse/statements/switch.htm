SKETCH OF HOW RESEARCH MIGHT CONTINUE AND RESULTS BE PRESENTED

JScript Statement: switch

Syntax

switch ( expression ) { [[ case-statement ]...] [ default-statement ] [[ case-statement ]...] }

The expression in parentheses is required. The required braces may enclose in any order at most one default-statement and any number of case-statements. Each case-statement has the form

case expression : [ statement-list ]

The default-statement has the form

default : [ statement-list ]
